# mia-onnx
Modules to run ONNX deep learning models in MIA

## Installation
Download mia-onnx-x.x.x.jar file from [Releases](https://github.com/mianalysis/mia-onnx/releases) and put in the "jars" folder of your copy of Fiji.  MIA should automatically detect "Run ONNX model" module.

## Usage
This module has been tested on ONNX models generated by Arivis Pro.  It will automatically tile the image (with a user-specified tile overlap to help reduce discontinuities) and can run on both RGB and greyscale images.  Stacks and timeseries will be processed slice-by-slice.
